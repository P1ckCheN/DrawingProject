## Compile and Run
- You can
1. Click the project 

> DawingProject.vcxproj

2. Run it with debug version in Win32 platform

> Debug + x86

- Or
1. Click the .exe file in folder

> freepainting.exe

## Operation
- Move mouse and push left button to draw. if you finish, release it.
- Click right button to erase the board.
- Have a good time!
